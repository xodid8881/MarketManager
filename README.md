# MarketManager

- 서버 상 플레이어 거래소 활성화 플러그인
- 플러그인 사용간 부가 플러그인

- InventoryLib 플러그인을 이용해야 합니다.
- https://github.com/xodid8881/InventoryLib
- 추가적으로 머니 플러그인이 존재한다면 개인적 연동을 진행해야 함을 당부합니다.
